***TO START LOCALLY:***

1. Clone repo
   ```console
   git clone https://github.com/NastyaNev/test-chart.git
   ```

2. Install dependencies from package.json
   ```console
   npm install
   ```
   
3. Start project
   ```console
   npm run dev
   ```
